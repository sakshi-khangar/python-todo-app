# python-todo-app
- A simple and beginner-friendly Console-Based To-Do List Application built using Python.
- This project allows users to add, view, remove, and save tasks using text-file storage, making the to-do list persistent even after closing the program.

## Features
- Add tasks
- View all tasks
- Remove tasks by number
- Automatically saves tasks in tasks.txt
- Simple & clean Python code

## Technologies Used
- Python
- File Handling (open, read/write)
- Lists & String Manipulation

## How It Works
- Displays menu (View / Add / Remove / Save & Exit)
- Tasks are stored in a Python list
- On exit, tasks are written to tasks.txt
- On restart, tasks are loaded again automatically
